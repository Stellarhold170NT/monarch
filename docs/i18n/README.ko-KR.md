코드베이스를 완전히 장악하는 고급 AI 에이전트 시스템. 당신이 명령하고 코드베이스가 적응하는 빠른 워크플로우를 제공합니다.

> 비싸고 단일한 모델을 빌리는 것은 그만. *Solo Leveling*에서 영감 받은 **Monarch**는 절대적인 권위를 의미합니다.
>
> 거대한 단일 시스템이 아니라, 당신의 명령대로 움직이는 **그림자 군단**——전문 에이전트들이 필요합니다.
>
> 터미널에서 단 한 번의 명령을 내리면, 오픈소스 생태계 전체가 정렬되어 코드베이스를 정복합니다.

---

## 활성화 상태

| 상태 | 설명 |
|-------|-------------|
| **Monarch** | 저장소 완전 통제. 높은 수준의 의도를 전달하면 Monarch가 실행 경로를 조율합니다. |
| **Ruler** | 엄격한 코드 아키텍처, 시스템 설계 규칙, 깔끔한 디커플링을 강제합니다. |
| **System** | 복잡한 기능 요청을 원자적 작업으로 분해하고 자동 실행합니다. |
| **Quicksilver** | 컨텍스트 윈도우와 토큰 스트림을 최적화하는 초고속 실행 엔진. |

---

## 그림자 군단 (서브 에이전트)

| 에이전트 | 역할 | 프로필 |
|-------|------|---------|
| **Igris** | 아키텍트 | 정밀함, 엄격한 논리 검증, 깔끔한 리팩토링에 집중합니다. |
| **Beru** | 자가 치유 | 런타임 실행을 모니터링하고, 에러 로그를 스캔하여 자동으로 버그를 수정합니다. |
| **Greed** | 제거 엔진 | 데드 코드, 보일러플레이트, 기술 부채를 무자비하게 삭제합니다. |
