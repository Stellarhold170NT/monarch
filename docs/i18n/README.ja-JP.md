コードベースを完全に制御する高度な AI エージェントシステム。あなたが指示し、コードベースが適応する高速ワークフローを実現します。

> 高額で単一のモデルをレンタルするのはもう終わり。*Solo Leveling* に触発された **Monarch** は絶対的な権威を意味します。
>
> 必要なのは、肥大化した単一システムではなく、あなたの命令通りに動く**影の軍団**——特殊エージェントたちです。
>
> ターミナルからひとつの命令を発するだけで、オープンソースエコシステム全体が連携し、コードベースを征服します。

## 1. アクティベーション状態

| 状態 | 説明 |
|-------|-------------|
| **Monarch** | リポジトリを完全に制御。高レベルの意図を伝えれば、Monarch が実行経路を編成します。 |
| **Ruler** | 厳格なコードアーキテクチャ、システム設計ルール、クリーンな疎結合を強制。 |
| **System** | 複雑な機能要求をアトミックなタスクに分解し、自動実行。 |
| **Quicksilver** | コンテキストウィンドウとトークンストリームを最適化する高速実行エンジン。 |
