## 1. Activation States

| State | Description |
|-------|-------------|
| **Monarch** | Total repository control. You provide the high-level intent; Monarch orchestrates the execution path. |
| **Ruler** | Enforces strict code architecture, system design rules, and clean decoupling. |
| **System** | Deconstructs complex feature requests into atomic tasks and automatically drives execution. |
| **Quicksilver** | High-speed execution engine optimizing context windows and streaming tokens with minimal latency. |
